
This app compares the rate of change of two stocks. This works with any ticker symbol found on yahoo finance. 
The purpose of this app is to use it as a tool to see if a stock is outpacing the market or a specific sector or competitor


How to use app
1.  Run window.py to start program
2.  stock one is the stock that you want to compare the growth of stock two. For example,
    if I want to compare Apple to the S&P 500 I would put apple in stock one and SPY in stock two
3.  For candle history you can enter 1y, 5y
4.  For candle size put 1d
5.  Hit Graph button
